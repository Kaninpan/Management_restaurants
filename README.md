# Management_restaurants
## โปรแกรมจัดการร้านอาหาร ด้วยโปรแกรม Netbeans
    ปี 3 ตอนกำลังศึกษา ได้ทำ Project โปรแกรมจัดการร้านอาหาร ด้วยโปรแกรม Netbeans 
    โปรเจคนี้เป็นโปรแกรมจัดการร้านอาหารแบบเดสก์ท็อป พัฒนาด้วยภาษา Java 
    โดยเน้นการเลือกเมนูอาหาร คำนวณราคา รวมยอด ออกใบเสร็จ และคำนวณเงินทอนภายในหน้าจอเดียว 

## เทคโนโลยีที่ใช้

- `Java SE 8`
- `Java Swing`
- `AWT`
- `NetBeans IDE`
- `Ant` ผ่านไฟล์ `build.xml`
- `JAR` สำหรับ build และนำไปรัน
- `AbsoluteLayout` และ `Beans Binding`

## ความสามารถของโปรแกรม

- เลือกเมนูอาหารหลักหลายรายการ
- เลือกของทานเล่นและเครื่องดื่ม
- คิดค่าจัดส่งที่บ้าน
- รวมราคาทั้งหมดอัตโนมัติ
- ออกใบเสร็จในหน้าจอโปรแกรม
- คำนวณเงินที่ลูกค้าจ่ายและเงินทอน
- มีเครื่องคิดเลขในตัว

โปรแกรมนี้เป็น `Desktop Application` ไม่ใช่เว็บแอป และไม่มีการใช้ฐานข้อมูลมาเชื่อมต่อกับระบบ ข้อมูลเมนู ราคา และการคำนวณต่าง ๆ ถูกกำหนดไว้ในโค้ดของโปรแกรมโดยตรง
